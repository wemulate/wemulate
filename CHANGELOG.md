## 1.0.0 (released 2021-09-16)

Initial release.

The following features are included:
* Add connection
* Add parameter
* Set parameter
* Delete parameter
* Delete connection
* Reset connection
* Reset device
* Show connections

The following parameters are included:
* Delay
* Bandwidth
* Jitter
* Packet loss
